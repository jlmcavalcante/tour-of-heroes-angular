Gerar componentes através do Angular Schematics: Botão direito na pasta

Criar módulo de rotas: ng g module app-routing --flat --module=app
  --flat: Evita criar nova pasta para armazenar o novo módulo, mantendo-o na principal.
  --module=app: Informa para que o modulo seja registrado dentro do módulo app.

Deleta linha de código no vscode: ctrl +shift + k

ng generate component dashboard --skip-tests
  - Cria componente sem arquivo de testes
